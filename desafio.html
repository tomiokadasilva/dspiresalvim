Desafio CSS - Layout Responsivo de Caixas

Objetivo:

Implementar um layout responsivo utilizando os conceitos do box-model em CSS para criar uma estrutura de caixas que se ajuste de forma agradável em diferentes tamanhos de tela.

Instruções:

  1. Crie um arquivo HTML básico com a estrutura inicial.
  2. Utilize o CSS para estilizar o layout conforme as especificações abaixo.

Especificações do Layout

  • Crie uma caixa principal (div) com uma largura máxima de 800 pixels e centralize-a na tela.
  • Dentro da caixa principal, crie três caixas filhas (div) lado a lado.
  • As caixas filhas devem ter uma largura de 30% da largura da caixa principal.
  • Entre as caixas filhas, adicione uma margem de 10 pixels.
  • Cada caixa deve conter algum conteúdo, como texto ou imagens, para que seja visível o posicionamento.

Desafio Extra:

  • Faça com que o layout seja fluido, ajustando-se suavemente à medida que a largura da tela muda.
  • Adicione uma borda de 2 pixels a todas as caixas.
  • Utilize cores de fundo diferentes para cada caixa, de forma a destacar visualmente a separação entre elas.
  • Adicione algum efeito de transição (como hover) para as caixas, como mudança de cor ou sombra.

Dica: Use as propriedades box-sizing, margin, padding, width, max-width, border, e display para implementar o box-model de forma eficiente.

Esse desafio ajudará a praticar a aplicação prática dos conceitos do box-model em CSS, ao mesmo tempo em que permite explorar a responsividade do layout. Boa codificação!
